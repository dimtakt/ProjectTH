리소스로 사용된 이미지들은 모두 원본 대비 2배 업스케일링을 하였습니다.
(사용하지 않은 일부 리소스들은 작업이 안돼있을 수 있습니다.)


또한 확장자가 없는 파일의 파일명을 보면 A_B_XCYD_E 꼴로 적혀 있을텐데, 각각

A는 스프라이트 요소 하나의 가로 길이 (업스케일링 전 원본 기준),
B는 스프라이트 요소 하나의 세로 길이 (업스케일링 전 원본 기준),
C는 해당 스프라이트 가로에 요소가 몇 개씩 있는지 (없으면 1),
D는 해당 스프라이트 세로에 요소가 몇 개씩 있는지 (없으면 1),
E는 해당 스프라이트 전체의 요소 갯수가 몇 개인지 (없으면 C or D와 동일)

를 뜻합니다.

예시..
64_64_X3Y3_8 : 가로64 세로64 가로3칸 세로3칸 격자에 8개의 요소가 있는 스프라이트.
128_64_Y4 : 가로128 세로64 가로1칸 세로4칸 격자에 4개의 요소가 있는 스프라이트.
